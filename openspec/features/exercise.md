# Feature: Exercise System

## 1. Exercise Types
- **Multiple Choice:** Chọn nghĩa đúng của từ.
- **Fill in the blank:** Điền chữ cái còn thiếu để hoàn thiện từ.
- **Listening:** Nghe âm thanh và chọn từ đúng.

## 2. Logic chấm điểm
- Mỗi câu đúng: +10 điểm.
- Hoàn thành bài tập với >80% câu đúng: Đánh dấu bài học là "Completed".
- Lưu kết quả vào Local DB để vẽ biểu đồ tiến độ.
